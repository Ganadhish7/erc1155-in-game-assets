# ERC-1155 contract

This is a smart contract that implements the assets in a game using tokens.
Two fungible currencies (gold and silver), and two fungible weapons(sword and sheild).
One non-fungible token (crown).

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat ignition deploy ./ignition/modules/Lock.js
```
